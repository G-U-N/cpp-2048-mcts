# 2048-AI



### 基于C++面向对象编程的linux终端小游戏。

- [x] 支持人类玩家！

- [x] 支持智能Agent！
  - [x]  随机Agent
  - [x]  RandomMCTSAgent(使用启发式的状态奖励函数与随机的Rollout策略)
- [x] 设置不同的随机种子可能会有不同的效果
  - [x] 有些种子可以成功合成2048
  - [ ] 有些则只能合成到1024，有待改进。
